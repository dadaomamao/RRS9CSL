- `AI.list`
- `eSIM-AI.list`
- `GoogleAccountAiUS.list`
- `ProxyLite.list`
- `Direct.list`
- `FinanceRealDNSDirect.list`
- `HighPriorityCNRealDNSDirect.list`
- `AppleDownloadProxy.list`
- `AppleBrowseHKSmart.list`
- `AppleMiscDirect.list`
- `crypto.list`
- `Dropbox.list`
- `AGENTS.md`

一般说明文档统一放在 `docs/`。这样做是为了减少自动读取流程把说明性 Markdown 当作规则输入的机会。

`AGENTS.md` 留在根目录，因为它是全仓库最高维护规则，不是规则列表。任何维护者或自动化在修改本仓库任何文件前，必须先读取根目录 [AGENTS.md](../AGENTS.md)。
